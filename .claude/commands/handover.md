# 🏁 Handover to Architect

모든 구현이 끝난 후 아키텍트에게 작업물을 넘기기 전 무결성을 점검합니다.

## 📋 절차

1. **최종 계획 확인**: `implementation_plan.md`의 **모든 체크박스가 `[x]`인지** 전수 확인합니다. 미완료(`[ ]`) 항목이 하나라도 있으면 즉시 중단하고 사용자에게 알립니다.

2. **에비던스 확인**: `.claude/evidence/` 폴더에 관련 증거 파일(로그, 테스트 결과 등)이 모두 생성되었는지 확인합니다.

3. **핵심 변경사항 보고**: 구현 중 발생한 특이사항 및 아키텍트가 참고할 변경 포인트를 요약합니다.

4. **안티그래비티 호출 안내**:
   > "모든 구현과 로컬 검증이 완료되었습니다. 이제 **안티그래비티(Gemini)**를 불러 `@folder`와 에비던스 파일을 기반으로 `walkthrough.md`를 작성해달라고 요청하세요."
