# 🔍 Performance Verification Protocol

### 1. 실행 쿼리 분석 (Execution Log)
- 클로드는 코드를 실행하거나 테스트를 수행할 때 발생하는 하이버네이트/SQL 로그를 샅샅이 분석한다.
- **체크리스트**: 
  - 의도치 않은 SELECT 쿼리가 반복적으로 발생하는가? (N+1 체크)
  - 페이징 시 전체 카운트 쿼리가 매번 나가는가?

### 2. 증거(Evidence) 기록
- 분석 결과는 `.claude/evidence/{branch}.md`에 다음과 같이 기록한다.
  - **최적화 전/후 비교**: 발생한 쿼리 개수 변화 (예: $1+N$개 → 1개).
  - **로그 캡처**: 핵심 쿼리문과 실제 파라미터가 바인딩된 로그의 일부.
  - **성공 판정**: 안티그래비티가 `walkthrough.md`를 쓸 때 성능 개선을 확신할 수 있는 구체적인 수치.
